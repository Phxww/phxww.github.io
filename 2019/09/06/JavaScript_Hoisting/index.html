undefined 與 not defined

在了解 Hoisting 之前，我們先稍微了解一下 undefined、not defined 的差別：

  • undefined : 記憶體已有準備位置，但還未定義型態，或是沒有指定的屬性
  • not defined : 還沒準備空間
